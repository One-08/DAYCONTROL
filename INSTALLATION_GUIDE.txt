═══════════════════════════════════════════════════════════════════════════
                    TO RUN ON YOUR LOCAL MACHINE
═══════════════════════════════════════════════════════════════════════════

STEP 1: Install Node.js (One-time setup)
────────────────────────────────────────
  Option A - Using official installer:
    → Download from: https://nodejs.org/
    → Get LTS version (recommended)
    → Run installer and follow steps
    → This will also install npm automatically

  Option B - Using system package manager:
    macOS (Homebrew):
      $ brew install node

    Ubuntu/Debian:
      $ sudo apt-get update
      $ sudo apt-get install nodejs npm

    Windows:
      → Use Windows Package Manager or installer

STEP 2: Download/Copy the Project
─────────────────────────────────
  The entire project is at: /home/vish/snap/DayControl/
  
  Copy all files to your desired location:
    $ cp -r /home/vish/snap/DayControl ~/my-projects/daycontrol

STEP 3: Navigate to Project
────────────────────────────
  $ cd ~/my-projects/daycontrol
  
  Or if on same system:
    $ cd /home/vish/snap/DayControl

STEP 4: Install Dependencies
──────────────────────────────
  $ npm install
  
  This will:
  - Read package.json
  - Download 37 npm packages
  - Create node_modules folder
  - Take ~2-5 minutes

STEP 5: Configure Google Sign-In
──────────────────────────────────
  A. Get Google Client ID:
    1. Go to https://console.cloud.google.com/
    2. Create new project (or use existing)
    3. Enable Google+ API
    4. Go to Credentials
    5. Create OAuth 2.0 credentials → Android
    6. Add your signing key SHA-1
    7. Copy the Client ID

  B. Configure .env file:
    $ cp .env.example .env
    
    Edit .env and update:
      GOOGLE_CLIENT_ID=YOUR_GOOGLE_CLIENT_ID.apps.googleusercontent.com

STEP 6: Run the App!
────────────────────

  Option A - Start Development Server:
    $ npm start
    
    Then choose platform:
      - Press 'a' for Android
      - Press 'i' for iOS (Mac only)
      - Press 'w' for Web browser

  Option B - Run on Android Directly:
    $ npm run android
    (Make sure Android emulator is running)

  Option C - Run on Web Browser:
    $ npm run web
    (Opens automatically in browser)

═══════════════════════════════════════════════════════════════════════════
                      TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Issue: "Command not found: npm"
Solution: Node.js/npm not installed. See STEP 1 above.

Issue: "Module not found errors"
Solution: Run 'npm install' again to reinstall dependencies

Issue: "EACCES permission denied"
Solution: Use 'sudo' or check folder permissions

Issue: "Port 19000 already in use"
Solution: Wait a moment or use different port: expo start --port 19001

Issue: "Google Sign-In fails"
Solution: Verify Client ID in .env matches your Google Console config
